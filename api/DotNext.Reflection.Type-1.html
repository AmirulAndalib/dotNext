Equals

Provides smart equality check.

Declaration
public static readonly Operator<T, T, bool> Equals
Field Value
Type Description
Operator<T, T, System.Boolean>
Remarks

If type T has equality operator then use it. Otherwise, for reference types, this delegate always calls System.Object.Equals(System.Object, System.Object) method. For value type, it calls equality operator or System.IEquatable<T>.Equals(T) if it is implemented by the value type; else, it calls BitwiseEquals(T, T).

Type Description
Operator<T, System.Int32>
Remarks

For reference types, this delegate always calls System.Object.GetHashCode() virtual method. For value type, it calls System.Object.GetHashCode() if it is overridden by the value type; otherwise, it calls BitwiseHashCode(T, Boolean).
