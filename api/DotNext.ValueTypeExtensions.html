Obtain a value of type TO by reinterpreting the object representation of FROM.

Type Description

Conversion result.
Type Parameters
Name Description
FROM

The type of input struct.
TO

The type of output struct.
Remarks

Every bit in the value representation of the returned TO object is equal to the corresponding bit in the object representation of FROM. The values of padding bits in the returned TO object are unspecified. The method takes into account size of FROM and TO types and able to provide conversion between types of different size.
