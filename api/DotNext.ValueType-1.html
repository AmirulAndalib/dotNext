Obtain a value of type TO by reinterpreting the object representation of T.

public static void BitCast<TO>(in T input, out TO output)

    where TO : struct
Parameters
Type Name Description
T input

A value to convert.
TO output

Conversion result.
Type Parameters
Name Description
TO

The type of output struct.
Remarks

Every bit in the value representation of the returned TO object is equal to the corresponding bit in the object representation of T. The values of padding bits in the returned TO object are unspecified. The method takes into account size of T and TO types and able to provide conversion between types of different size.
