Class AsyncDelegate

Provides set of methods for asynchronous invocation of various delegates.

Namespace: DotNext.Threading

Syntax
public static class AsyncDelegate
Remarks

BeginInvoke and EndInvoke methods of delegate type are supported only in .NET Framework. This class provides alternative approach which allows to invoke delegate asynchronously with full support of async/await feature.
