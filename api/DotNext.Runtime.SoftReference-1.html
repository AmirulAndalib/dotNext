Class SoftReference<T>

Represents a form of weak reference which is eligible for garbage collection in Generation 2 only.

public sealed class SoftReference<T> : IOptionMonad<T>, ISupplier<object> where T : class
Type Parameters
Name Description
T

The type of the object referenced.
Remarks

The object that is soft-referenced survive garbage collection in Generation 0 and 1. When GC is triggered for Generation 2, the object can be reclaimed according to SoftReferenceOptions. All public instance members of this type are thread-safe.
